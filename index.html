Tethered-Monocopter Downwash Gantry

Raspberry Pi · Arduino · LQR @ 50 Hz · Time-of-flight sensing

A two-dimensional test rig for studying downwash. An upstream propeller generates a time-varying wake, and a tethered monocopter riding on linear rails downstream measures the altitude disturbance it causes. A 50 Hz LQR loop keeps the monocopter vertically stabilised while data is collected.

Raspberry Pi/Python control, Arduino serial I/O, ESC PWM, stepper actuation and time-of-flight sensing run in a single real-time stack. The rig produced the hardware dataset behind the paper below on memory-based wake prediction.

Real-Time Reinforcement-Learning Marble Labyrinth

DreamerV3 · YOLOv5 · Arduino · 45 Hz control loop

An online reinforcement-learning system that learns to play a wooden marble labyrinth — the classic toy where two knobs tilt the board to steer a marble past holes. Stepper motors bolted to the knobs actuate the board, a camera with YOLOv5 tracking follows the marble, and DreamerV3 learns on the physical hardware from scratch.

A 3D-printed re-loader returns the marble after each attempt so training runs unsupervised, and the training loop is patched so gradient updates only run between physical episodes, preserving the 45 Hz control deadline during live training.

Wake Up to the Past: Using Memory to Model Fluid Wake Effects on Robots

L. Vendruscolo, E. Sebastián, A. Prorok, A. Shankar. We show that agile wake effects on robots cannot be accurately predicted from instantaneous state alone, and that giving models memory of past states cuts prediction error across simulation and real hardware — including on the gantry above. arXiv:2603.22472↗
